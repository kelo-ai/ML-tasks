## 🔹 TASK 1: Iris Dataset Analysis

### Objective
To analyze and visualize the Iris dataset and understand feature relationships.

### Dataset
- Source: sklearn built-in dataset  
- Features: Sepal length, Sepal width, Petal length, Petal width  
- Target: Species  

### Models Used
- Logistic Regression (basic classification)

### Results
- High classification accuracy achieved  
- Petal features are most important  

### Key Insights
- Setosa is clearly separable  
- Slight overlap between Versicolor and Virginica

## 🔹 TASK 3: Heart Disease Prediction

### Objective
Predict whether a person has heart disease using medical data.

### Dataset
- Source: UCI Heart Disease Dataset (Kaggle)  

### Models Used
- Logistic Regression  

### Evaluation Metrics
- Accuracy  
- Confusion Matrix  
- ROC-AUC Curve  

### Results
- Achieved accuracy: ~85–90%  
- Important features: Chest pain, cholesterol, max heart rate  

### Key Insights
- Model performs well for binary classification  
- Useful for early risk detection
